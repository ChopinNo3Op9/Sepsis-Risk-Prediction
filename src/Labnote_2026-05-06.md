## 3. Dataset / Split
- Data: PhysioNet 2019 (`training_setA` + `training_setB`)
- Patients loaded: 220
- Features: 20 clinical variables (vitals + key labs)
- Windowing: history=24h, forecast=6h
- Split: sequential window split 70/15/15
- Positive rate: 2.10% overall

## 6. Interpretation
- Many additional complex models were tested, but none exceeded the existing TCN+BiGRU+Transformer.
- The key likely reason is temporal structure: flattening windows for tree/MLP models loses explicit sequence dynamics.
- Extremely low thresholds (often near 0) indicate heavy class imbalance pressure; this keeps sensitivity higher but hurts specificity.
- Ensemble voting did not improve test AUROC over the best single tree model.
