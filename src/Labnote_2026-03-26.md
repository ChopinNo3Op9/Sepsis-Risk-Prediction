# Sepsis Experiment Notes

## Date
- 2026-03-26

## 1. Research Objective
- Research Objective: evaluate single-model Gradient-Boosted Trees for first sepsis onset in next 6 hours using 24-hour history and document bootstrap performance.

## 2. Data Source
- PhysioNet Challenge 2019 (public ICU challenge training data)
- URL root: https://physionet.org/files/challenge-2019/1.0.0/training/
- Folders used: training_setA and training_setB
- Local data path: data/raw

## 3. Research Background and Theoretical Basis
- Sepsis is acute life-threatening response to infection; early detection is critical.
- Physiological vitals from PhysioNet provide temporal signals with missingness patterns.
- Gradient-boosted trees on flattened 24h time blocks provide a nonlinear tabular baseline to compare against logistic regression.

## 4. Experimental Procedures
- Preprocess: read p*.psv files, causal forward fill, median imputation (train median) for remaining NaNs.
- Features: HR, O2Sat, Temp, SBP, MAP, DBP, Resp, FiO2
- Label: 1 if first sepsis onset occurs within (t, t+6] after 24h history window.
- Split: 70/15/15 sequence-based (with 1867/400/401 samples from 2668 total windows).
- Model (with tuning): XGBClassifier(objective='binary:logistic', eval_metric='logloss', tree_method='hist', scale_pos_weight=neg/pos).
- Hyperparameter tuning: validation-grid search (24 trials), selection by validation AUPRC with tie-breakers AUROC then lower Brier.
- Tuning grid:
  - n_estimators: [150, 250, 350]
  - learning_rate: [0.03, 0.05]
  - max_depth: [3, 4]
  - min_child_weight: [1, 3]
  - subsample: [0.8]
  - colsample_bytree: [0.8]
  - reg_lambda: [1.0]
- Best tuned parameters: n_estimators=350, learning_rate=0.03, max_depth=4, min_child_weight=3, subsample=0.8, colsample_bytree=0.8, reg_lambda=1.0.
- Bootstrap metrics on test: 100 rounds.

## 6. Analysis and Conclusion
- Conclusion: after parameter tuning, validation AUROC/AUPRC improved slightly, but test sensitivity remained zero at threshold 0.5, indicating thresholding/calibration dominates performance gap.
- Limitations: only 220 patients, simplified split strategy, fixed threshold=0.5, and no probability calibration.
- Next steps: tune decision threshold on validation set (optimize recall/F1), calibrate probabilities (Platt/isotonic), and evaluate patient-level split to reduce optimistic temporal bias.
